open the app in the dist. folder, (wifi_scan.exe)

enter you gateway(router's Ip address), you can find this using the 
ipconfig/all command in cmd then the last section for WIFI

now start scanning, it will alert you of unknown devices on the network

Add MAC to allow a device on your network by adding the name and the MAC address of the device you want to add, remove to no allow
